# qr-code-generator
Simple qr-code generator

Сделано специально для генерации ссылки на сайтик с указанием текущей версии игры.

```
Usage: zx-qr-code [OPTIONS] --url <URL>
```

```
Options:
  -o, --output <FILE>
          Output filename
          
          [default: qr-code.bin]

  -u, --url <URL>
          Encoded url

  -s, --size <SIZE>
          QR-code canvas size
          
          [default: 32]

  -h, --help
          Print help (see a summary with '-h')

  -V, --version
          Print version

```
